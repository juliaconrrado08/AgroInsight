# AgroInsight
O site atua como um hub de inteligência focado em resolver duas das maiores dores atuais do produtor: o endividamento elevado (com o consequente aumento nos pedidos de recuperação judicial) e a vulnerabilidade pela dependência de insumos importados. A abordagem é prática, oferecendo desde análises de mercado até soluções de crédito e insumos.
🛒 Pokémart.com
Bem-vindo ao repositório do Pokémart.com, uma plataforma de comércio eletrônico fictícia do mundo Pokémon onde treinadores de todas as regiões podem comprar e vender itens essenciais para suas jornadas, como Pokébolas, Potions e Berries!

Este projeto foi desenvolvido como um exercício prático para consolidar fundamentos de HTML5 e CSS3.

🚀 Tecnologias Utilizadas
O projeto foi construído utilizando as tecnologias web fundamentais:

HTML5: Estruturação semântica da página (títulos, parágrafos, imagens, botões e links de navegação).

CSS3: Estilização visual (definição de cores de fundo, fontes, tamanhos e espaçamentos).

📂 Estrutura do Projeto
O projeto segue a estrutura padrão de desenvolvimento web:

Plaintext


├── index.html          # Página principal e ponto de entrada do site
├── style.css           # Arquivo de estilização com as regras CSS
└── imagens/            # Pasta contendo os assets visuais (ex: logo-pokemart.png)
🛠️ Funcionalidades Aplicadas
Durante o desenvolvimento deste projeto, foram aplicados os seguintes conceitos:

Estrutura Semântica: Uso correto de tags como <h1> para títulos importantes e <p> para descrições de texto.

Acessibilidade em Imagens: Inclusão correta do atributo alt nas imagens (<img>) para leitores de tela.

Diferenciação Semântica: Aplicação prática de tags <a> para navegação entre páginas e <button> para ações interativas (como adicionar itens ao carrinho).

Estilização Customizada: Customização do plano de fundo (background-color), cores de texto (color), além de ajustes de tipografia com font-family e font-size.
